# 文档导航

这个目录保留的是适合放在 GitHub 仓库中的公开文档，重点服务于以下场景：

- 快速了解项目能做什么
- 本地启动和体验系统
- 配置外部模型接口
- 迁移或部署到另一台电脑

## 推荐阅读顺序

1. [使用说明](guides/使用说明.md)
   适合先了解页面结构、基本操作和创作流程。
2. [接入生图 API 说明](guides/接入生图API说明.md)
   适合配置图片 Provider 和测试接口。
3. [部署与迁移说明](guides/部署与迁移说明.md)
   适合准备迁移、单入口运行和部署前检查。
4. [接口示例](guides/api-examples.http)
   适合本地调试和接口验证。

## 仓库文档原则

GitHub 仓库优先保留以下内容：

- 能帮助别人快速理解项目的说明
- 能帮助别人运行项目的指南
- 能帮助别人部署项目的资料

以下内容默认本地保留，不作为公开仓库主内容：

- 内部版本总结
- 过程性复盘
- 本地交付用 Word 成品
- 临时调试产物和运行时数据
