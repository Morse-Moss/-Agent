# 使用说明

## 启动顺序

1. 启动后端：`start-backend.cmd`
2. 启动前端：`start-frontend.cmd`
3. 可选冒烟验证：`smoke-test.cmd`

## 登录

- 用户名：`admin`
- 密码：`admin123`

## 创作流程

1. 进入创作台。
2. 上传白底产品图。
3. 输入主图需求、卖点和风格。
4. 系统会先追问 1-2 个关键问题，或按默认方案生成。
5. 在作品详情页继续修改、审核、定稿或派生新版本。

## 接入真实生图 API

1. 进入“系统设置”。
2. 保存 `Image API Key`。
3. 保存 `Image Provider / Image API URL / Image Model / Image API Key Header`。
4. 点击“测试生图 Provider”确认连通。

## 文档重新生成

运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build-docs.ps1
```
